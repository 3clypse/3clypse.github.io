My projects in a nutshell

Hi everyone, I have finished all my github projects except the Raspberry Pi Printer Kiosk. Now, I will write a little about the others:


Milanuncios

Milanuncios.com is a website where you can sell anything, but it has a inconvenient:

The free version allow 1 posting reload each 24h, so each morning you should login and renew that post.

My automated version allow you to refresh it without any kind of problems.

Thanks to my friend Jesus Gomez, we have a final & stable release.


Flask Tweets Locator

It's a simple mini website where you can search by tweets.

I made this at the university, but I liked how it worked and I decided to upload it to github.


POC Raspberry Pi Trojan

In this POC I wanted to show how easy is introduce a mini-pc (Trojan) running a reverse SSH that allow connections from outside.

I have this POC in spanish, but I cant send it in english.
